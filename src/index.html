Questions fréquentes

Dois-je lier mon compte bancaire ?
Non, absolument pas. Divided sert uniquement de calculatrice intelligente pour savoir qui doit quoi. Les remboursements se font en dehors de l'application (via virement bancaire, Lydia, Payconiq, ou en cash).
L'application est-elle disponible sur mobile ?
Oui ! Vous utilisez actuellement la version Web qui s'adapte parfaitement à l'écran de votre smartphone. Une application iOS et Android native est également en cours de développement.
Que faire si quelqu'un a payé pour une seule personne ?
Cette fonctionnalité avancée (répartition inégale) est disponible dans le module d'ajout de dépenses. Vous pourrez bientôt décocher les participants qui ne sont pas concernés par un achat spécifique.
